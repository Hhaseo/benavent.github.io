Gestion Multi-Langues

Outil de gestion de fichiers de langues en .xml.

En cours de rédaction. Il s'agit d'un outil de gestion de base de données pour des fichiers langues (actuellement sauvegardés au format XML, mais avec pour but de proposer d'autres formats). Cet outil prend la forme un client C++/QT permettant de manipuler aisément cette base de données pour les traducteurs, accompagné de différents scripts permettant le chargement des fichiers de langues dans vos applications (actuellement il y a un module C# et C# dédié à Unity 3D).
